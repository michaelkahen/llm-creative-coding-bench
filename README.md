# LLM Creative Coding Playground

A small collection of creative-coding experiments made by asking different
language models to build standalone HTML sketches from the same prompt.

> **Just for fun — this is not a real benchmark.**
> I’m simply playing around with different models, prompts, and coding
> environments to see the kinds of visual experiments they produce. The files here
> are not a rigorous evaluation, a leaderboard, or a measure of any model’s overall
> quality.

## What’s here

Each project directory contains a prompt (`.txt`) and one or more
model-generated, self-contained HTML implementations:

- **ASCII Twirl** — ASCII-art text with a pseudo-3D twirl.
- **Advanced Water Simulation** — an interactive canvas water simulation.
- **Digital Ecosystem** — autonomous creatures, food, and emergent behavior.
- **Fluid Text** — particles that assemble into text.
- **Infinite Terrain** — scrolling procedural ASCII terrain.

Open any `.html` file directly in a modern browser to try it. They are intended
to be dependency-free, single-file experiments.

## How to read the names

HTML filenames include the model and/or environment used to produce that version.
Multiple files in a folder generally mean different attempts at the same
creative-coding prompt—not ranked results or controlled comparisons.

## Notes

These are casual experiments. Prompts, model settings, tools, and runs can vary,
so please don’t treat differences between outputs as scientific conclusions. If you
have ideas for a fun prompt or model to try, feel free to open an issue or send a
pull request.
